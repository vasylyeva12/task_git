Задание 1. 1. Создаем в своем аккаунте новый репозиторий, ставим галочку на файл Readme. Копируем ссылку в закладке-SSH 2. Клонируем на локальный диск в компьютер репозиторий командой git clone ссылка-SSH.

Задание 2. 1. В клонированной папке репозитория на локальном диске создаём необходимые файлы, или проект. 2. В терминале в папке с клонированным репозиторием последовательно выполняем команды: - git add . - добавление всех файлов (папок) - git commit -m "Message" - фиксируем изменения с комментарием - git push - пушим файлы с изменениями на github

Задание 3. 1. Для получения изменений из github.com на локальный компьютер необходимо перейти в папку с репозиторием и выполнить команду git pull.

Задание 4. git status - отображает состояние рабочего каталога и раздела проиндексированных файлов, показывает неотслеживаемые файлы git log - перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке — последние коммиты находятся вверху. git remote -v - проверка связи репозитория с github (удалённая ссылка на данный репозиторий)
